Сборка использует Silex как микрофреймворк (управление путями) и Twig как шаблонизатор


Структура папок:

/css - css файлы ВАШИХ стилей

/js - js файлы ВАШИХ собственных скриптов. Модули которые Вы скачиваете должны быть в vendor

/img - тут ПАПКИ с картинками. не надо пихать все картинки в одну папку

/pages - тут шаблоны html 
/vendor/ - тут лежат папки подключенных модулей

/vendor/autoload.php - файл где подключаются классы модулей

/index.php - главный файл на php

/.htaccess - файл - инструкция для APACHE где должны быть указания редиректа  на index.php Иначе не сработают виртуальные пути
